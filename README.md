# Zomato Delivery Operations Dashboard

Power BI dashboard built for the Analyst Hive community challenge.

## Overview
Analysed Zomato delivery operations data covering 45,584K total orders 
across multiple cities in India, tracking delivery performance, 
vehicle types, and personnel ratings.

## Key Insights
- Motorcycles dominate deliveries at 57.98% of all vehicle types
- March recorded the highest order volume at 22K orders
- Average delivery person rating of 4.63 out of 5
- Snack orders lead at 25.29% of total order types

## Tools Used
Power BI · DAX · Power Query · Maps Visual · Data Modelling
